# Norvo

> Bring neighbors closer, block by block

Norvo is a hyperlocal community platform designed to connect local neighborhoods through trusted communication, local buy-and-sell, lost & found, community events, and real-time neighborhood updates.

## Features

- **Neighborhood Feed** - Stay updated with what's happening in your community
- **Local Marketplace** - Buy and sell within your neighborhood
- **Lost & Found** - Help reunite lost items with their owners
- **Community Events** - Discover and participate in local events
- **Trusted Communication** - Connect with verified neighbors

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **TDesign** - UI component library
- **Lucide React** - Icons

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd Norvo

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Open http://localhost:5173 in your browser
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
